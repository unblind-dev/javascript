## Prerequisites

- At least one ingested tenant with metrics
- An [API key](https://console.unblind.dev/settings)
- 
## Getting Started

1. Add your tenant ID and API key.

```bash
TENANT_ID=<your_tenant_id>
UNBLIND_API_KEY=<your_api_key>


2. Run
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
